# MouseFox

MouseFox works like a GUI launcher for pgnet games using
[kvex](https://github.com/ArielHorwitz/kvex), an extension for kivy.

## Built-in examples
Tic-tac-toe:
```python3
import mousefox

mousefox.examples.tictactoe()
```

Text chat:
```python3
import mousefox

mousefox.examples.chat()
```
